Week 6 - Part 1
Procedural checkerboard texture on a rectangle (repeat + nearest)
Please use a browser that supports HTML5 canvas.
What you should see
A rectangle “floor” with a black/white checkerboard texture. The pattern repeats 4× across width and 10× along length, using repeat addressing and nearest filtering.
Tip: run a local server so shader.wgsl loads (Live Server or python3 -m http.server).
